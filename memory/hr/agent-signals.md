# agent-signals.md — Work Agent 能力缺口信号

由 HR 每日采集写入（/hr-daily-signal）。保留最近 30 天，更早的自动删除。

## 信号类型

| 类型 | 说明 |
|------|------|
| 任务失败 | 任务未完成、需要重试、被退回 |
| 重复卡点 | 同类问题在多个会话中反复出现 |
| 边界混淆 | agent 接手了不属于其职责的任务 |
| 评审官打回 | auditor 标记了该 agent 的交付质量问题 |
| 用户反馈 | 用户直接指出某 agent 的不足（权重最高） |

## 格式

```
- [YYYY-MM-DD] [agent-id] 信号类型: 描述
```
