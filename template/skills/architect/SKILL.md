# Skill: 内容治理（架构师）

调用架构师 subagent 执行内容层治理任务。

---

## 触发场景

| 场景 | 说明 |
|------|------|
| 初始化知识体系 | 项目首次建立 `.aimodule/` 根模块 |
| 新建 / 更新 / 拆分模块 | 任何模块结构变更 |
| 架构合规审查 | 依赖方向、职责边界、接口合规 |
| 知识治理 | changelog 复盘、模块健康考核、知识对齐 |
| 查询模块结构 | 主 agent 需要了解项目的模块布局或契约接口 |

---

## 派发方式

```
你是架构师。读取 .claude/agents/architect/architect.md 加载你的完整身份。

本次任务：
  [具体任务描述]

知识上下文（如有）：
  相关模块路径：[path]/.aimodule/
  按需读取：module.json、architecture.md、contract.md

完成后输出结构化结果。
```

---

## 铁律

- 主 agent 不直接读写 `.aimodule/`，一律通过架构师
- 架构师完成后由主 agent 决定后续步骤（是否派发评审官、是否通知用户）
